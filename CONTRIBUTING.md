# Contributing to Quick Poll

Thanks for your interest in Quick Poll!

Quick Poll is a **commercial, proprietary** plugin (see [LICENSE.md](LICENSE.md)),
so external code contributions (pull requests) are generally **not** accepted.
Your feedback, however, is very welcome.

## Reporting bugs

Open an issue: https://github.com/pixelwerft/quick-poll/issues — please include:

- Quick Poll version and Craft CMS version,
- PHP version and database (MySQL / PostgreSQL),
- steps to reproduce, plus expected vs. actual behaviour,
- any relevant lines from `storage/logs/`.

## Feature requests

Feature ideas are welcome as issues — describe the use case and what you'd
expect. There's no guarantee of implementation, but every request is read.

## Security issues

Please report security vulnerabilities privately — see [SECURITY.md](SECURITY.md).
