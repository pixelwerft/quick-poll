# Security Policy

## Supported versions

The latest released version of Quick Poll receives security updates.

| Version | Supported |
|---------|-----------|
| 1.1.x   | ✅        |
| < 1.1   | ❌        |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report them privately by email to **manuel@pixelwerft.ch**, or via GitHub's
[private vulnerability reporting](https://github.com/pixelwerft/quick-poll/security/advisories/new).
Please include:

- the Quick Poll version and Craft CMS version,
- a description of the vulnerability and its impact,
- steps to reproduce (a proof of concept if possible).

You'll receive an acknowledgement within a few business days. Once confirmed and
fixed, a patched release is published to the Craft Plugin Store and credited in
the changelog — unless you prefer to remain anonymous.
